4. ファイルフォーマット¶

abICSの入力ファイルは, 以下の４つのセクションから構成されます.

  1. [replica] セクション
レプリカ数や温度の幅, モンテカルロステップ数など,レプリカ交換モンテカルロ部分のパラメータを指定します.
  1. [solver] セクション
ソルバーの種類 (VASP, QE, …)、ソルバーへのパス、不変な入力ファイルのあるディレクトリなど（第一原理計算）ソルバーのパラメータを指定します.
  1. [observer] セクション
取得する物理量の種類などを指定します.
  1. [config] セクション
合金の配位などを指定します.
